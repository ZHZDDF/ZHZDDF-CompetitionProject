# ZHZDDF大学生计算机博弈大赛比赛工具使用说明

## 下载链接

https://github.com/ZHZDDF/ZHZDDF-CompetitionProject/releases/download/1.0.0/ZHZDDF-DHBDTool.exe

## 程序概述

此程序是zhzddf制作的参加大学生计算机博弈大赛的比赛工具。

## 功能特性

### 棋种选择

• 配备了五子棋、围棋和爱恩斯坦棋

• 在"开始"栏可切换不同棋种

### 脚本管理

在"设置"栏可添加或切换脚本：
• 点击"添加"即可添加一个脚本

• 脚本配置步骤：

<img width="275" height="136" alt="image" src="https://github.com/user-attachments/assets/f38f7692-c3dc-458e-a021-e3829d4d5073" />

  1. 选择要使用的脚本
  2. 选择脚本位置
  3. 选择脚本运行环境位置
  4. 删除脚本（可选）

### 对局功能

基本信息填写

• 对手信息、比赛地点、比赛名称

• 在打谱前随时可填写或修改

爱恩斯坦棋对局流程

准备阶段

• 赛前选择先后手（红色方固定先手）

• 按双方预设拖动棋子摆子

• 点击"开始"按钮进入游戏阶段

游戏阶段

• 每一步前设置色子数或按"掷色子"按钮随机生成色子数

• 我方轮次：按"执行"按钮运行脚本自动下子

• 对方轮次：拖动棋子帮对手下棋

结束阶段

• 当一方获胜后，点击"打谱"按钮将棋谱保存在本地

• 点击"重开"回到准备阶段

19路围棋对局流程

准备阶段

• 选择先后手

• 按"开始"进入游戏阶段

游戏阶段

• 我方轮次：按"执行"落子

• 对方轮次：帮对手落子

结束阶段

• 当一方获胜后，点击"打谱"按钮将棋谱保存在本地

• 点击"重开"回到准备阶段

五子棋对局流程

准备阶段

• 选择先后手

• 设置五手N打的具体数目

• 点击"开始"开始游戏

游戏阶段

• 五手N打阶段：落子方能连下N颗子

• N打方选择要移除的子

• 选择完后点击"确认选择"按钮

结束阶段

• 当一方获胜后，点击"打谱"按钮将棋谱保存在本地

• 点击"重开"回到准备阶段

### 棋谱功能

• 点击"棋谱"按钮可随时查看棋谱目前状态

• 棋谱格式同中国大学生计算机博弈大赛官方要求
